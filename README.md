# Проект: Место

### Описание проекта
* Лого
* Профиль с аватаром, личной информацией и возможностью её редактирования
* Фото галерея

**Использованные технологии**

Этот проект написан в качестве домашней практической работы №4 в результате прохождения курса Яндекс.Практикума по веб-разработке. Для вёрстки применялись grid и flex, относительные величины, метатеги, что позволило достичь эффекта адаптивности. Также реализовано диалоговое окно для редактирования данных профиля в рамках практики использования основ JavaScript.

**Планируемая будущая оптимизация**

* Доработать "резиновость", применяя больше относительных величин
* Сделать так, чтобы данные формы сохранялись на сервер
